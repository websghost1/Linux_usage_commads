במערכות לינוקס ישנן כמה דרכים לבדוק מהו סוג הזיכרון (RAM) המותקן. הנה מספר פקודות נפוצות:

---

## 1. שימוש ב־dmidecode

פקודה זו מציגה מידע מפירמות ה־DMI/SMBIOS של המערכת, כולל על מודולי הזיכרון:

```bash
sudo dmidecode -t memory
```

- תוצאות הפקודה כוללות מידע על כל סלוט זיכרון, כולל **סוג (DDR2 / DDR3 / DDR4 וכו')**, **גודל (Size)**, **מהירות (Speed)**, **יצרן (Manufacturer)** ועוד.  
- אם תרצה לחפש נקודתית את סוג הזיכרון, אפשר להשתמש ב־grep:

```bash
sudo dmidecode -t memory | grep -i "Type:"
```

---

## 2. שימוש ב-lshw

כלי נוסף להצגת חומרה במערכת:

```bash
sudo lshw -C memory
```

התוצאה תציג פרטים על הזיכרון במערכת, כולל **bank**, **description**, **size** ועוד.

---

## 3. שימוש ב-inxi (אם מותקן)

אם המערכת כוללת את הכלי inxi (לא מותקן בכל הפצות כברירת מחדל), אפשר:

```bash
inxi -m
```

ולקבל מידע מקוצר על הזיכרון (RAM) במערכת.

---

## דוגמה לפלט צפוי (dmidecode)


כאן ניתן לראות:
- **Type**: DDR4  
- **Speed**: 2667 MHz  
- **Manufacturer**: Kingston  

---

### סיכום
- **הפקודה העיקרית**: `sudo dmidecode -t memory`  
- **חלופה**: `sudo lshw -C memory`  
- כך תוכל לדעת האם הזיכרון הוא DDR3, DDR4, DDR5, מהירותו, כמות, היצרן ועוד.
